# Official Repository for RallyToFlag

## Tech Stack:
- Laravel 9
- MySQL (MariaDB 10)
- Vue 3
- Inertia
- Tailwind

## Installation

```
git clone git@github.com:rallytoflag/rallytoflag.git

cd rallytoflag

composer install

npm install
```

## Running the application

```
php artisan serve

npm run dev
```
