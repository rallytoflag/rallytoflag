# Official Repository for RallyToFlag

## Tech Stack:
- Laravel 9
- MySQL (MariaDB)
- Vue 3
- Inertia
- Tailwind
- PHP 8.1
- Node 16

## Installation

```
git clone git@github.com:rallytoflag/rallytoflag.git

cd rallytoflag

composer install

npm install
```

## Running the application

```
php artisan serve
```

```
npm run dev
```
